Etiquetas Semanticas Estructurales

Esto es una etiqueta contenedora que no tiene semantica
header es una cabecera de un sitio web o una seccion
main define la seccion principal del documento, solo puede existir una etiqueta main por documento
footer es el pie de pagina de un sitio web o de una seccion
nav representa una navegacion
Article representa una seccion de autocontenido (por si sola entiendes le contenido sin tener la interfaz del sitio, la parte donde va todo el articulo del blog)
aside representa contenido complementario o secundario, podriamos meter alguna lista de articulos relacionados, para en los sitios web donde metemos publicidad etc
section representa una seccion de contenido generico
address representa una direccion fisica si lo quieren ver asi o informacion de contacto

Etiquetas de bloque vs Etiquetas de linea

La etiqueta de linea por excelencia es span
